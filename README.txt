Application for adding contact information of participants
Чтобы запустить приложение локально, откройте терминал Bash  в каталоге, где находится `app.py`.
Затем включите виртуальную среду, используя следующие команды:

python -m venv virtual
source virtual/bin/activate

Установить flask, если не установлен, запустить `pip install flask`
После этого  запустить `flask run`, и HTTP-сервер запустится.
Вводим 127.0.0.1:5000 в веб-браузере и переходим к веб-приложению.
